# Changelog

## v0.1.0 - 2026-06-04

Initial public release of Favicon Forge.

### Added

- Local browser-based favicon generation.
- PNG outputs for favicon, Apple Touch Icon, and PWA icon sizes.
- Transparent, white, and custom background modes.
- Square, soft rounded, and circular icon shapes.
- Adjustable icon padding.
- Copyable HTML icon tags.
- Downloadable `site.webmanifest`.

### Notes

- The first version outputs PNG files only.
- `.ico` export and ZIP download are planned follow-up features.
