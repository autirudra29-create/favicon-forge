# 🛠️ favicon-forge - Create website icons on your computer

[![](https://img.shields.io/badge/Download-Latest_Release-blue.svg)](https://github.com/autirudra29-create/favicon-forge/releases)

Favicon Forge helps you turn any logo or image into a set of icons for your website. Browsers and mobile devices need these small images to show your brand in tabs and on home screens. This tool runs entirely on your computer. It does not send your files to a server. You keep full control over your images.

## 🚀 Getting Started

Follow these steps to get the app running on your Windows computer.

1. Visit the [releases page](https://github.com/autirudra29-create/favicon-forge/releases) to find the latest version.
2. Look for the file ending in `.zip` under the Assets section.
3. Click the file name to start the download.
4. Open your Downloads folder.
5. Right-click the downloaded folder and choose Extract All.
6. Follow the prompts to unzip the files.
7. Open the folder once extraction finishes.
8. Locate the file named index.html.
9. Double-click index.html to open the tool in your web browser.

## ⚙️ How It Works

This tool processes images within your browser. It uses the Canvas API to resize and format your files. Because it runs locally, you do not need an internet connection to generate your icons.

The process follows three simple stages:

1. Upload your source file. Choose a high-quality logo or image for the best results.
2. Adjust your settings. Choose your shape, background color, and padding.
3. Download the result. The tool provides a folder containing all necessary sizes.

## 🎨 Features

This tool builds the specific files required for modern web standards.

*   Browser support: It generates standard 16x16, 32x32, and 48x48 pixel icons.
*   Apple devices: It creates the apple-touch-icon file for iPhone and iPad home screens.
*   Progressive Web Apps: It exports 192x192 and 512x512 icons for Android.
*   Flexible backgrounds: Choose between transparent, white, or a custom color.
*   Shape selection: Choose between square, circle, or softly rounded corners.
*   Padding tools: Add space around your logo so it does not touch the edges.
*   Manifest support: Get a site.webmanifest file to declare your icon paths.
*   HTML snippets: Copy the code lines directly to add them to your website.

## 🔒 Privacy

Privacy matters when you work with brand assets. This tool processes every image on your local machine. Your browser does not send your logo to any cloud service. No company logs your activity or saves your images. You can disconnect your internet once the page loads and the tool will still work.

## 🛠️ Frequently Asked Questions

What image formats work best?
We recommend using PNG or SVG files for your logo. High-resolution source images produce better results when scaled down.

Do I need to pay for any licenses?
No. The software is free. Use it for personal or commercial projects without restrictions.

What if my icon looks blurry?
Small icons like 16x16 use limited pixels. Ensure your source image has clear edges and high contrast. Increase the padding if the design looks too crowded.

Can I save my configuration?
The browser uses local storage to remember your last settings. If you close the page and return, your settings remain intact.

Does this tool call an external server?
The code is self-contained. No network requests are made during the icons generation process.
