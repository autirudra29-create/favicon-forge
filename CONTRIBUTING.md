# Contributing

Thanks for helping improve Favicon Forge.

## Good First Contributions

- Add more icon presets.
- Improve generated manifest options.
- Add `.ico` or ZIP export.
- Improve accessibility and keyboard support.
- Improve mobile layout and browser compatibility.

## Development

This project intentionally has no build step.

1. Open `index.html` in a browser.
2. Upload a test image.
3. Generate icons and verify downloads.
4. Open a pull request with a clear description of the behavior change.

## Principles

- Keep the app free.
- Keep image processing local.
- Avoid unnecessary dependencies.
- Make the tool understandable within the first screen.
