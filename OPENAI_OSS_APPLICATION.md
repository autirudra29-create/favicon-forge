# Codex for Open Source Application Notes

This file is a draft to help explain why Favicon Forge is a suitable open source project for the Codex for Open Source program.

## Project Summary

Favicon Forge is a free, local-first web tool that helps developers generate favicon, Apple Touch Icon, and PWA icon assets without uploading images or using paid services.

## Why It Matters

Many small open source projects and personal websites need correct favicon and PWA icon assets, but existing workflows often require manual resizing, paid design tools, or uploading private brand assets to third-party services. Favicon Forge keeps the workflow simple, free, and privacy-friendly.

## Maintainer Role

I am the primary maintainer of this project. I built and deployed the initial version, maintain the roadmap, and plan to review issues, ship releases, and improve the tool based on community feedback.
